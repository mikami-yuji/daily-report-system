# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [Unreleased]

## [1.5.0] - 2026-01-16

### Added
- 2026年度版プロジェクト（pyro-eclipse-2026）の作成
- 2026年度版Excelファイル（営業日報シート）への対応確認
- 重点顧客マトリクスでの担当者フィルタリング機能
- 直送先を含む重点顧客の正確な表示・集計

### Changed
- 重点顧客数KPIを担当者でフィルタリングした数に変更
- 得意先一覧の重点顧客フィルタで直送先も正確に表示
- 分析ページの重点顧客マトリクスで「重点顧客」フラグをチェック

### Fixed
- 重点顧客の直送先が正しくフィルタリングされない問題を修正
- 担当者による重点顧客フィルタリングが効かない問題を修正

## [1.4.0] - 2025-12-22

### Added
- 売上分析機能
- 前年売上データの表示

### Fixed
- サイドバーナビゲーションのルーティング問題を修正
- 「クレーム対応」リンクがホームにリダイレクトされる問題を解消

## [1.3.0] - 2025-12-15

### Added
- モバイル対応（レスポンシブデザイン）
- カード表示モード
- PWA対応（オフラインキャッシュ）

### Fixed
- Excel解析時の「Bad compressed size」エラーを修正
- キャッシュクリア機能の追加

## [1.2.0] - 2025-12-08

### Added
- 外出時間の入力UI改善
- 満足度（ランク）フィールドの表示制御
- タイムライン表示モード
- 上長コメント機能

### Changed
- Turbopack設定の追加
- バックエンド設定の改善（config.jsonサポート）

### Fixed
- マージコンフリクトの解消（globals.css）
- 無限ループ問題の修正
- `_x000D_`改行変換の修正

## [1.1.0] - 2025-11-26
